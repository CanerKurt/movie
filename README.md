Neo4j Veritabanı üzerinden film arama, detaylarını görüntüleme ve bağlantılarını gösteren graf.json oluşturmaya yarar.
Neo4j örnek veritabanı olarak https://github.com/neo4j-graph-examples/movies/blob/main/scripts/movies.cypher kullanılmıştır.
Test veya film python dosyalarını çalıştırmak için veritabanının açık olması gerekmektedir.
Oluşan graf dosyası film.py veya test.py dizininde exports/ klasörüne çıkartılır.
